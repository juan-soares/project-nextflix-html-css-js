HTML

HTML (HyperText Markup Language) é utilizado para construção de páginas da web através de tags, tal como parágrafos, seções, títulos e divisões, que garantem a semântica e o sentido do conteúdo.

CSS

CSS (Cascading Style Sheets) é utilizado para estilização de páginas da web através de seletores e propriedades, tal como cores, fontes, tamanhos e estilos, que garantem a consistência visual e a adaptabilidade do conteúdo.

JavaScript

JavaScript é utilizado para interação de páginas da web através de manipulação dos elementos na tela, tal como ações após eventos do usuário, validações de formulários e animações, que garantem a experiência e a dinâmica com o conteúdo.

React

React é utilizado para construção e interação de páginas da web através de componentes, tal como atualizações isoladas, manutenção facilitada e compactação de código, que garantem a experiência e a dinâmica do usuário com o conteúdo.

Next.js

Next.js é utilizado para construção e interação de páginas da web através de componentes gerados no servidor, tal como carregamento de informações externas (APIS), manutenção facilitada e compactação de código, que garantem a performance e a Segurança do conteúdo.
